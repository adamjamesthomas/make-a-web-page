Web Basics

When you type an address into a web browser, your computer sends an HTTP request to the server. In response, the server sends back an HTML document. The web browser interprets the HTML document and shows you a nice, clean website.

HTML

HTML (Hyper Text Markup Language) consists of the content, or text, of a website as well as the markup, or format. HTML is written using tags and elements. These elements are either inline or block. Block elements appear on their own line with nothing beside them. There are many tags and elements, but below are a couple of the basics.

Bold

The <b> tag is used to make text bold, just like fresh mango is used to make a salsa bold.

Italics

You can use the <i> tag to italicize text. This makes your text slanted and therefore fancier.

Dumb Computers

Computers are dumb. But really, people are dumb for thinking computers will treat information the same way a human would. Computers have strict rules for how they interpret and run code, so the slightest mistake can break everything.

Structure

Boxes

A web page is made up of a bunch of boxes. Even circles are actually boxes at heart. One of the best ways to analyze the structure of a web page is to divide it into squares.

Div and Span

Div and span are two html tags that are useful for providing structure to your HTML. Though these tags don't change the loo of the page on their own, they divide the page into meaningful chunks that will be easier to edit/style later.

Cascading Style Sheets (CSS)

With a structured HTML document, it is very easy to style several elements on the page at the same time. By using element a class names, you can avoid repetition by defining the style for all similar elements at once.

Referencing from HTML

An HTML document can reference a CSS style by adding a link in the head portion of the HTML. That style will then be applied by the browser to the contents of the page.

Box Model

The CSS box model describes 4 parts of a box that you can style.

  • The margin, or blank space, around the box to keep it from bumping anything around it.
  • The border is the line drawn around the box.
  • The padding is the space between the border and where the content begins.
  • The content is the actual text or image that will reside in the box.
